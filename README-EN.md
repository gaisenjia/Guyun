# Guyun

An AI breeding and grain quality control front-end showcase.

This repository contains a static web page that presents a product concept for agricultural quality management. It focuses on grain sample collection, AI visual inspection, risk grading, inspection reports, batch traceability, and resource recycling.

## Languages

- [中文](README.md)
- English: current document
- [日本語](README-JP.md)

## Project Scope

The page includes the following sections:

- Hero section: introduces the concept of AI breeding, quality inspection, and batch traceability
- AI breeding workflow: shows the process from sample collection to industry application
- Batch traceability: simulates batch lookup and quality tracking information
- AI grain inspection report: simulates visual inspection results, risk levels, and handling recommendations
- Abnormal grain handling: presents abnormality detection, fermentation treatment, and farmland reuse
- Industry scenarios: shows potential applications in agriculture, food safety, and supply chain management

## How to View

This repository currently contains the static source files only. No separate online preview page is included in the repository.

To view the page:

1. Download or clone this repository.
2. Open `index.html` in a browser.

The project can also be deployed to a static hosting service such as GitHub Pages, Netlify, or Vercel.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- WebP image assets

## Highlights

- No framework dependency
- Responsive layout for desktop and mobile screens
- Scroll animations, button interactions, and simulated data cards
- Image assets are stored in the `assets/` directory

## Note

This is a front-end showcase project. The inspection results, batch information, and AI analysis shown on the page are demonstration data, not output from a real inspection system.
