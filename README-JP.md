# 谷蕴新生

AI 育種と穀物品質管理をテーマにしたフロントエンド展示プロジェクトです。

このリポジトリは静的な Web ページ作品です。穀物サンプルの収集、AI による画像認識、リスク評価、検査レポート、ロット追跡、資源循環などの場面を通して、農業品質管理プラットフォームのコンセプトを表現しています。

## 言語

- [中文](README.md)
- [English](README-EN.md)
- 日本語：現在のドキュメント

## プロジェクト内容

ページには以下の内容が含まれています。

- トップビュー：AI 育種、品質検査、ロット追跡の全体コンセプト
- AI 育種フロー：サンプル収集から産業応用までの流れ
- ロット追跡：ロット検索と品質追跡情報のデモ表示
- AI 穀物検査レポート：画像検査結果、リスクレベル、処理提案のデモ表示
- 異常穀物の処理：異常検出、発酵処理、農地への再利用
- 産業応用シーン：農業生産、食品安全、サプライチェーン管理への応用例

## 表示方法

このリポジトリには静的なソースファイルのみが含まれています。別のオンラインプレビューページは含まれていません。

表示するには：

1. このリポジトリをダウンロード、またはクローンします。
2. ブラウザで `index.html` を開きます。

GitHub Pages、Netlify、Vercel などの静的ホスティングサービスにデプロイして公開することもできます。

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- WebP 画像素材

## 特徴

- フレームワークに依存しない軽量な静的ページ
- デスクトップとモバイルに対応したレスポンシブレイアウト
- スクロールアニメーション、ボタン操作、デモ用データカード
- 画像素材はすべて `assets/` ディレクトリに保存

## 注意

これはフロントエンド展示用のプロジェクトです。ページ内の検査結果、ロット情報、AI 分析内容はデモデータであり、実際の検査システムの出力ではありません。
