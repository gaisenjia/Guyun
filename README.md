## �🌾 项目简介

本项目是一个面向谷物品质管控的前端展示平台，涵盖以下核心能力：

- **AI谷物识别** — 识别杂质、破碎粒、颜色异常、霉变风险等视觉特征
- **智能检测报告** — 生成综合评分、风险等级和处理建议
- **批次溯源** — 检测结果进入溯源链路，服务品质证明和批次流转

## 📋 管控流程

| 步骤 | 说明 |
|------|------|
| 01 原粮样本采集 | 采集示例谷物样本，形成标准化检测起点 |
| 02 批次信息录入 | 录入样本类型、批次编号和采样信息，建立批次档案 |
| 03 AI谷物识别 | 识别杂质、破碎粒、颜色异常、霉变风险等视觉特征 |
| 04 风险评级与检测报告 | 生成综合评分、风险等级和处理建议，输出品质判定 |
| 05 AI育种辅助 | 优质样本纳入候选池，为育种迭代提供数据支撑 |
| 06 供应链溯源与产业应用 | 检测结果进入溯源链路，服务品质证明和批次流转 |
