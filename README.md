# 谷蕴新生

AI 育种全链路品质管控前端展示项目。

本项目是一个静态网页作品，围绕谷物样本采集、AI 视觉识别、风险评级、检测报告、批次溯源和资源循环等场景，展示农业品质管控平台的产品概念。

## 多语言版本

- 中文版：当前文档
- [English](README-EN.md)
- [日本語](README-JP.md)

## 项目内容

网页主要包含以下模块：

- 首页展示：介绍“AI 育种、品质检测、批次溯源”的整体概念
- AI 育种流程：展示从样本采集到产业应用的流程
- 批次扫码溯源：模拟批次查询和质量追踪信息展示
- AI 谷物检测报告：模拟谷物视觉检测结果、风险等级和处理建议
- 异常谷物处理：展示异常识别、发酵处理和农田回用的资源循环路径
- 产业应用场景：展示平台在农业生产、食品安全和供应链管理中的应用方向

## 如何查看

这个仓库目前只包含静态网页源码，没有单独上传在线预览页面。

查看方式：

1. 下载或克隆本仓库到本地。
2. 用浏览器打开 `index.html`。

也可以把仓库部署到 GitHub Pages、Netlify、Vercel 等静态托管平台后在线访问。

## 技术栈

- HTML5
- CSS3
- 原生 JavaScript
- WebP 图片素材

## 项目特点

- 无框架依赖，适合作为轻量级静态网页展示
- 使用响应式布局，适配桌面端和移动端
- 使用滚动动画、按钮交互和模拟数据卡片增强展示效果
- 所有展示素材都放在 `assets/` 目录中

## 说明

这是一个前端展示项目，页面中的检测结果、批次信息和 AI 分析内容为演示数据，不代表真实检测系统输出。
